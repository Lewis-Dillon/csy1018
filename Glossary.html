The following are 50 terms used in web development

Term Definition
HTML "Hypertext markup language" is a language used to create websites. It involves using a combination of tags (opening and closing tags).
CSS "Cascading style sheet" is the method used to style a website (involves visuals such as fonts, colours, structure ect). Only one style sheet is needed to style multiple webpages.
PHP "Hypertext Preprocessor" allows webcontent to be desplayed and interacted with.
"ALT" Attribute Identified by the alt tag, this allows additional information to be desplayed about an image.
"Title" Attribute Normally on the top of the page, tends to describe the webpage and allow search engins to identify said page.
Call to Action Some element on the webpage (could be button, icon ect) that suggests the user to perform a certain action.
Promo Area An area on a webpage designed to sell a product or simply promote it.
Exit Page The last page a user will see before exiting the webpage.
Drop Down Normally associated with menus, allows the user to select sub options after a primary option. (example being a menu that, once hovered over, will provide additional options.
Rollover Once the mouse is hovering over an element (image, text ect) said element will change visually.
Input Field An area in which the user can input values.
Optimisation Reducing an elements size to allow a user a more user friendly experience.
White Space A term used for the space around webpage elements that could highlight a certain feature. White is not the only colour associated with white space.
Mock-Up A design example of priview of the website.
UI The "User Interface" is what the users sees and can interact with. the UI normally is what determins how a user will feel about the website.
UX The "User Experience" is how the user feels when using the website.
WireFrame A design on how the website will look. Nomrally created before coding and is a representation of the layout.
Address Bar The bar located at the top of the browser of which allows a user to input a website address.
ASP a coding langage normally used for windows servers.
XML "Extemsoble Markup Language" is used for creating custom markup languages.
404 a page that a user sees when they are unable to reach a page due to its deletion or non-existance.
Breakpoints Allows a webpage to dynamically change its layout to fit devices such as smartphones.
Cookies A small text file that tracks a users actions on the website.
CMS "Content Management Systems" is software that allows more control on the content of a website.
Domain What people would input into the browser to to visit the website.
Favicon An icon that identifies a website or company associated with said website.
IA "Information Architecture" is information about the navigation, content and design of a website.
Meta Tag A tag in the HTML that includes information about the website. Could include site desription of the website and who made it.
Navigation refers to methods of which allows a user to move between pages on a website.
Registrar A third party of which allow a website to be resgistered a domain name.
Site Map A map created in the design phase of the website. Involves navigation pathways and works in a hierarchical manner.
Slider Also known as a carousel, is a rotating sequence of images/text.
Banner Ad Advertisement placed on a website that links to a third party.
API "Application Programing Interface" allows other applicaions to communicate with the website.
WYSIWYG "What You See Is What You Get" is a term associated with software that gives a live preview of what is being coded.
URL "Uniform Resource Locator" allows a web developer to attach links to certain aspects of a website (text, images etc.)
301 Redirects Allows users to click on an old website and be automatically redirected to a new website created by the same company.
SEO "Search Engine Optimisation" refers to ensureing that your website will appear when it is searched for in a browser. (example being adding keywords to describe the website).
Flyout Navigation A menu that contains many links to pages on the website. Can be identified via multiple columns.
Style Guide A document that helps a developer design how the page will look. Normally contains colour pallets, fonts and images.
.png "Portable Network Graphics" An image file of which uses lossless compression.
.gif "Graphics Interchange Format" uses lossless compression and can be animated.
.swf "Small Web Format" a file of which can display animation on the web. cannot be edited without original file.
Responsive Design Designing method that involves tackling hard issues first to allow easy development later.(example being starting with the mobile design first).
Backend What the users cannot see but is availible for administrators to update the website.
SERP The page that appears after searching for websites on a browers
Front End Normally containing the UI, the front end is what visitors to a website will see and is visually effected through the CSS.
Valid These are websites and webpages that have no errors. Normrally done through putting code through validators.
WWW "World Wide Web" relates to the internet of which a website will reside.
Traffic relates to the amount of users that will enter and exit a website at any given time.
